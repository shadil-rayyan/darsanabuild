# Introduction

Welcome to the documentation for the CMNS & Scholarship Management System.

## What is CMNS?

CMNS stands for **CMNS & Scholarship Management System**, a comprehensive tool for managing academic and scholarship-related data.

## Features

- **User Management**: Manage students, faculty, and administrative users.
- **Scholarship Tracking**: Track scholarships, applications, and awards.
- **Reporting**: Generate detailed reports on various aspects of the system.

For more information, please refer to the [System Overview](/system-architecture) page.
